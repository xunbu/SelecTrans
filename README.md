# SelecTrans
[项目主页](http://www.xunbu.cc/lightrans)  
一个windows端划词翻译软件  
可以做到网页、pdf等多格式划线翻译  
内置百度、有道等多种翻译引擎
OCR使用百度OCR需注册API账号（详见项目主页）
支持输入翻译  
可以去除换行符复制粘贴PDF文本
![Image text](https://github.com/xunbu/SelecTrans/blob/main/examplephoto/%E5%8A%9F%E8%83%BD%E4%BB%8B%E7%BB%8D.jpg)
快捷键 清空页面：ctrl+delete，调节字体：ctrl+↑/↓
